# Wumpus Project

This is an Angular project that utilizes Node.js.

## Usage

1. Make sure you have Node.js installed.
2. Run the following command to install the dependencies:

```
   npm install
```
3. Run the following command toset up and start the application in development mode:

```
ng serve -o
```

Then, open your browser and navigate to http://localhost:4200/ to see the application in real-time.

## Notes

This project uses Angular for web development.
